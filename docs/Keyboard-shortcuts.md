# Keyboard Shortcuts

| Key combination                                            | Description         |
|------------------------------------------------------------|---------------------|
| `Ctrl+r` `Cmd+r` `F5`                                      | Reload current tab. |
| `Ctrl++` `Cmd++` <br /> `Ctrl+ScrollUp` `Cmd+ScrollUp`     | Zoom in.            |
| `Ctrl+-` `Cmd+-` <br /> `Ctrl+ScrollDown` `Cmd+ScrollDown` | Zoom out.           |
| `Ctrl+0` `Cmd+0`                                           | Reset zoom.         |
